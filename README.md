# Strongdm Tap

## How do I install these formulae?

`brew install strongdm/tap/<formula>`

Or `brew tap strongdm/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "strongdm/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
